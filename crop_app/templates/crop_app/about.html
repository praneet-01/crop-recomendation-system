About CropSense

A final year engineering project in AI-powered precision agriculture

🎯 Project Objective

CropSense is a machine learning-based web application that helps farmers and agricultural professionals select the most suitable crop based on real-time environmental and soil conditions. By leveraging a Random Forest classifier trained on thousands of crop data points, the system provides accurate recommendations instantly.

🤖 Machine Learning Model

The system uses a Random Forest Classifier trained on the Crop Recommendation Dataset. The model achieved approximately 99% accuracy on test data, making it highly reliable for real-world use.

  • Algorithm: Random Forest (Scikit-learn)
  • Features: N, P, K, Temperature, Humidity, pH, Rainfall
  • Output: One of 22 crop types
  • Training accuracy: ~99%
